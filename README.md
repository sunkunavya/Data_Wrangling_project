# Data Immersion & Wrangling Project

## Objective
To clean and prepare raw customer data for analysis by handling missing values,
duplicates, inconsistent formats, and performing feature engineering.

## Tools Used
- Python
- Pandas
- NumPy

## Workflow
1. Data loading and profiling
2. Data quality assessment
3. Missing value treatment
4. Duplicate removal
5. Feature engineering
6. Exporting cleaned dataset

## Output
Cleaned and analysis-ready customer dataset.
